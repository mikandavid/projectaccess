# Website Feedback From Screenshot Review

Source: WhatsApp screenshots from May 7, 2026. This document combines the visible page context with the reviewer annotations so the feedback is actionable. Original wording is kept where useful, including typos, but grouped by page and section.

## Cross-Site / Global Feedback

### Primary CTA and Application Flow

Visible issue: Several buttons appear to point users toward the general Project Access International website or a generic "start" flow.

Reviewer feedback:

- The top-right `Kostenlos starten` button should be the Bootcamp application CTA.
- It should not send users to the Project Access International website.
- Applicants should apply through a form.
- The international website link should exist somewhere else, but less prominently.
- Project Access International should not be the main next step. It is only relevant for users who specifically want to learn more about the wider organization.

Implementation implication:

- Make the Bootcamp application / signup form the primary CTA across the site.
- Move the Project Access International link into a lower-priority location, such as the "About Project Access" section or footer.
- Check all application-related buttons, especially `Kostenlos starten`, `Als Schüler:in starten`, and `Zur Anmeldung`.

### Button Hierarchy and Colors

Visible issue: Primary and secondary buttons use different color treatments in ways that feel inconsistent.

Reviewer feedback:

- "hier und überall, es gibt nicht wirklich einen grund dass die beiden andere farben haben oder?"
- Suggested direction: primary action button can stay black filled and secondary buttons black outlined, or the top action button can be orange and these can be black.

Implementation implication:

- Define one consistent button hierarchy across the whole site.
- Use one primary style for main conversion actions.
- Use one secondary outline style for supporting actions.
- Avoid random color differences between paired buttons.

### Blue Image Treatment

Visible issue: Multiple large images have a blue overlay/tint.

Reviewer feedback:

- "wieso blue?"
- On the home hero: "wieso blue? außerdem nichtssagendes foto, lieber eins wo man alle sieht bei BC"
- On the partners page: "wieso blau?"

Implementation implication:

- Remove or reduce the blue overlay on photos.
- Prefer warm, authentic Bootcamp photography that fits the site's cream/orange visual system.
- Use photos where students/participants are visible and the moment is meaningful.
